# tech-blog
cms style blog site where developers can publish posts and comment on each others posts.

## Table of Contents
* [Installation](#installation)
* [Usage](#usage)
* [Technologies](#technologies)

## Installation
To install necessary dependencies, run the following command:
```
npm i
```

## Usage
```
npm start
```
